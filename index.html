🔗 Data Connection Mastery

Learn to connect and import data sources

0/4 Steps
1

Open Power BI Desktop

Launch Power BI Desktop and start a new project

📋 Instructions:

  1. Open Power BI Desktop on your computer
  2. If prompted, select "Get data" or create a blank report
  3. You should see the main Power BI interface with ribbon menu
Tip: Make sure you have the latest version of Power BI Desktop for the best experience.
2

Import CSV Data

Connect to your downloaded sample data file

📋 Instructions:

  1. Click "Get Data" in the Home ribbon
  2. Select "Text/CSV" from the list
  3. Browse and select your downloaded "sample-data.csv" file
  4. Preview the data and click "Load"
Expected Result: You should see your data table appear in the Fields panel on the right.
3

Add Second Data Source

Import the recruitment metrics data

📋 Instructions:

  1. Click "Get Data" again
  2. Select "Text/CSV"
  3. Choose "recruitment-metrics.csv"
  4. Load the data
Challenge: Can you identify the relationship between the two datasets?
4

Verify Data Import

Check that all data is properly loaded

📋 Instructions:

  1. Look at the Fields panel on the right
  2. Verify you see both data tables
  3. Expand each table to see the columns
  4. Check data types and column names
Congratulations! You've mastered data connection and earned the Data Master badge!

📊 Visualization Mastery

Create stunning charts and visual elements

0/5 Steps
1

Create a Column Chart

Build your first visualization

📋 Instructions:

  1. Select "Clustered Column Chart" from visualizations panel
  2. Drag "Department" to Axis
  3. Drag "Employee Count" to Values
  4. Observe the chart creation
2

Add a Pie Chart

Show data distribution with pie chart

📋 Instructions:

  1. Click empty space in report canvas
  2. Select "Pie Chart" from visualizations
  3. Add "Gender" to Legend
  4. Add "Employee Count" to Values
3

Create KPI Cards

Display key metrics prominently

📋 Instructions:

  1. Select "Card" visualization
  2. Add "Total Employees" to Fields
  3. Create another card for "Average Salary"
  4. Position cards at the top of your report
4

Add a Line Chart

Show trends over time

📋 Instructions:

  1. Select "Line Chart" from visualizations
  2. Add "Date" to Axis
  3. Add "Recruitment Count" to Values
  4. Observe the trend line
5

Create a Data Table

Display detailed information in tabular format

📋 Instructions:

  1. Select "Table" visualization
  2. Add "Employee Name", "Department", "Salary" to Values
  3. Resize the table appropriately
  4. Check data formatting
Amazing! You've earned the Visual Expert badge!

🎨 Design & Formatting Mastery

Make your dashboard look professional

0/4 Steps
1

Apply Color Theme

Choose consistent colors for your dashboard

📋 Instructions:

  1. Select a visual (click on it)
  2. Go to Format pane (paint roller icon)
  3. Expand "Data colors"
  4. Choose a consistent color scheme
2

Add Titles and Labels

Make your visuals clear and informative

📋 Instructions:

  1. Select each visual
  2. In Format pane, turn on "Title"
  3. Edit title text to be descriptive
  4. Adjust font size and color
3

Arrange Layout

Organize visuals in a logical layout

📋 Instructions:

  1. Drag and resize visuals
  2. Place KPI cards at the top
  3. Arrange charts in a grid below
  4. Leave appropriate spacing
4

Final Polish

Add background and finishing touches

📋 Instructions:

  1. Click empty space in report
  2. Format pane → Page background
  3. Choose subtle background color
  4. Ensure all text is readable
Excellent! You've earned the Design Guru badge!

Add Department Slicer

  4. Test the date filtering

Test Interactivity

  2. Adjust the date range
  3. Click on chart elements for cross-filtering
  4. Verify all visuals update correctly
Outstanding! You've earned the Interactive Pro badge!

Save Your Work
